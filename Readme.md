
HOW TO DEVELOP:
1. "cd frontend"
2. "npm run dev"
3. open page on "localhost:9002" 


HOW TO COMPILE FOR BUILD:
1. "cd frontend"
2. "npm run compileStatic"
3. your compiled files ready to deploy will be in "/frontend/dist"
